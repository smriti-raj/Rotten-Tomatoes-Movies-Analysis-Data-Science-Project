🎬 Rotten Tomatoes Movies Analysis 📌 Project Overview

This project performs an end-to-end Exploratory Data Analysis (EDA) and Machine Learning workflow using the Rotten Tomatoes Movies dataset. It explores movie ratings, genres, critics' reviews, audience scores, runtimes, studios, and release dates to uncover trends and insights. The project demonstrates data cleaning, visualization, feature engineering, statistical analysis, and predictive modeling using Python.

📈 Key Insights
Audience and critics' ratings often differ across genres.
Some studios consistently produce highly rated films.
Runtime has a limited relationship with movie ratings.
Drama and Comedy dominate the dataset by volume.
Higher Tomatometer ratings generally correspond to stronger audience reception.
